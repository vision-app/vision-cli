## Install

```shell

```

## Quick Start

## Commands

- `vusion help`: Show help of all commands

- `vusion app dev`
- `vusion app build`
- `vusion ui dev`
- `vusion ui build`

- `vusion -V, --version`: Show the version of current CLI
